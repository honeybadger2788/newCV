Play & Joy - E-commerce

Ver Proyecto

Experiencia basada en el proyecto integrador realizado en Digital House.

Se desarrolló desde el análisis de mercado pasando por el diseño y maquetado de la apariencia del sitio, hasta el diseño de la base de datos y la validación de los mismos.

Para el front se utilizó HTML, CSS y javascript para agregar dinamismo. Por el lado del back, se trabajó con Express dentro de Node. Se diseñó la BD usando MySQL y Sequelize como ORM.

Además, se insertó el consumo de API's externas para el formulario de registro; se usaron cookies para el guardado de los datos de login del usuario; a través del desarrollo de middlewares, se restringieron los accesos a los diferentes perfiles de usuarios; y para el carrito de compras, se almacenaron los datos utilizando localstorage.
